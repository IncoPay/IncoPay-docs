<Info>
  **Prerequisites**: Node.js 19+ and this `incopay-docs` folder with `docs.json`.
</Info>

## Preview docs locally

<Steps>
  <Step title="Install Mintlify CLI">

```bash
npm i -g mint
```

  </Step>
  <Step title="Run the dev server">

From the **`incopay-docs`** directory (where `docs.json` lives):

```bash
cd incopay-docs
mint dev
```

Open **http://localhost:3000** to view the docs. MDX changes hot-reload.

  </Step>
</Steps>

## Custom port

```bash
mint dev --port 3333
```

## Validate links

```bash
mint broken-links
```

## Working on IncoPay itself

These docs describe **IncoPay** (private, session-based x402 on Solana). The code is split across three repos:

1. **Client SDK** — [`solana-x402-sessions`](https://www.npmjs.com/package/solana-x402-sessions) on npm. Source at [`IncoPay/solana-x402-sessions`](https://github.com/IncoPay/solana-x402-sessions).
2. **Facilitator** — [`IncoPay/inco-facilitator`](https://github.com/IncoPay/inco-facilitator). Express server; runs locally with `npm run dev`.
3. **Demo app** — [`ayushsingh82/IncoPay`](https://github.com/ayushsingh82/IncoPay). Next.js app that consumes the SDK.

For Mintlify theme and navigation, see [Mintlify documentation](https://mintlify.com/docs).
