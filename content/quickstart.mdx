The fastest path: clone IncoPay, install, and run the Next.js app. The hosted facilitator is already live on Railway and the SDK is on npm — you don't run Kora, the facilitator, or any infra locally.

## Prerequisites

- **Node.js** 18+ and **Yarn** (or npm)

## Step 1: Clone and install

```bash
git clone https://github.com/IncoPay/IncoPay
cd IncoPay
yarn install
```

The published [`solana-x402-sessions`](https://www.npmjs.com/package/solana-x402-sessions) SDK installs as a regular dependency.

## Step 2: Run the app

```bash
yarn dev
```

That's it. Open [http://localhost:3000](http://localhost:3000).

The app comes pre-wired to the hosted facilitator at `https://inco-facilitator-production.up.railway.app` and the default devnet mint on Solana — no `.env.local` required for the demo path.

### Payment asset (x402 & demos)

Session flows, slot, and single-pay routes all price and settle in **USDC** (6 decimals). Transfers use **IncoToken + Inco Lightning** for confidential balances on the default mint. The same x402 stack also supports **standard SPL USDC**: set `NEXT_PUBLIC_TOKEN_MINT` (and match your facilitator’s `TOKEN_MINT`) to the USDC mint you want on devnet.

## Step 3: Try the demo

- **Home** — [http://localhost:3000](http://localhost:3000)
- **Sessions chat** (sign-once, settle 0.5 USDC per message) — [http://localhost:3000/sessions](http://localhost:3000/sessions)
- **Slot** (sign-once, settle 0.10 USDC per pull) — [http://localhost:3000/slot](http://localhost:3000/slot)
- **Single payment** (Pay 1 USDC) — [http://localhost:3000/started](http://localhost:3000/started)

Connect a Solana wallet on devnet (Phantom, Solflare, Backpack), click **Mint 100 USDC** to seed your IncoAccount, then **Authorize** a session and start chatting / pulling.

## Self-hosting (optional)

If you want to run the facilitator yourself, see [Hosted Stack](/docs/reference/hosted-stack) for the env vars needed by [`IncoPay/inco-facilitator`](https://github.com/IncoPay/inco-facilitator). It does not depend on Kora — the facilitator's own funded keypair pays the SOL fee for every settle.
